📡 How to set up NFC scanning

1
Add your inventory items using the Add Item button. Each item gets a unique ID automatically.
2
Copy the NFC URL shown next to each item in the inventory table.
3
Use an NFC writing app (e.g. NFC Tools on iOS/Android) to write the URL to an NFC tag sticker. Attach the sticker to your physical item or shelf.
4
When anyone taps the tag with an NFC-enabled phone, it opens this page and automatically logs the scan + decrements quantity by 1.
5
Bookmark this page URL to always have your live inventory dashboard available.
